# BP Measurement Tracker

This folder is ready for GitHub Pages.

1. Upload the complete folder contents to a GitHub repository.
2. In repository settings, enable **Pages** from the branch containing `index.html`.
3. Keep the `assets` directory unchanged. The app references the supplied images by their actual filenames and case:
   - `assets/LOGO.png`
   - `assets/NANA.jpeg`
   - `assets/MISC.webp`

The tracker submits data to the supplied Google Apps Script URL with `POST` and `mode: 'no-cors'`.
